This plugin displays a Bootstrap modal showing product details when a button is clicked. It uses a shortcode to output the button anywhere on your WordPress site.

== Description ==

**Product Details Modal** is a simple WordPress plugin that displays detailed information about a specific product (by product ID) inside a Bootstrap modal when a button is clicked.

You can easily change the product ID in the code or make it dynamic.  
The modal is responsive, user-friendly, and fully compatible with Bootstrap 5.

**Features:**
- Displays product details inside a Bootstrap modal
- Uses AJAX to fetch and show product details dynamically
- Easy to integrate using a shortcode
- Lightweight and developer-friendly
